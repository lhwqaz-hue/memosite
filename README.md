# 간편 메모 공유 웹사이트

PC와 모바일 간 간단한 텍스트 공유를 위한 웹사이트입니다. 로그인 없이 일회성 비밀번호로 메모를 공유할 수 있습니다.

## 주요 기능

- ✨ 4~12자 일회성 비밀번호 설정
- ⏱️ 1시간 동안 유효한 메모
- 💾 실시간 자동 저장
- 📱 PC와 모바일 간 간편 공유
- 🔒 로그인 불필요

## 사용 방법

### PC에서
1. 웹사이트 접속
2. 4~12자 사이의 일회성 비밀번호 입력
3. 메모 작성 (자동 저장됨)
4. 비밀번호를 모바일에 전달

### 모바일에서
1. 웹사이트 접속 후 같은 비밀번호 입력
2. 또는 `http://localhost:3000?p=비밀번호` 형태로 직접 접속
3. PC에서 작성한 메모 확인

## 설치 및 실행

### 필요 사항
- Node.js (v14 이상)

### 설치
```bash
npm install
```

### 실행
```bash
npm start
```

개발 모드 (자동 재시작):
```bash
npm run dev
```

서버가 시작되면 `http://localhost:3000`에서 접속 가능합니다.

## 기술 스택

- **백엔드**: Node.js, Express
- **프론트엔드**: Vanilla JavaScript, HTML, CSS
- **저장소**: 메모리 기반 (서버 재시작 시 초기화)

## 주의사항

- 메모는 서버 메모리에 저장되므로 서버 재시작 시 모든 데이터가 삭제됩니다
- 프로덕션 환경에서는 Redis나 데이터베이스 사용을 권장합니다
- 민감한 정보는 저장하지 마세요
- 메모는 1시간 후 자동으로 삭제됩니다
